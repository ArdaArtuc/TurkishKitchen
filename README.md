# TurkishKitchen
CS-210 Project

This project uses several data structures that were covered in class to simulate a small restaurant kitchen.  
Menu items, customer orders, recipe steps, and culinary stations are all managed by the application.

Description Of The Project 

This program models how a real kitchen operates.
Including:
- A menu of Turkish dishes
- A queue of customers
- A recipe tree
- Graph representing stations
- Searching and sorting

  To Compile & Run.
  Using an online compiler OR "g++ main.cpp -o kitchen"
